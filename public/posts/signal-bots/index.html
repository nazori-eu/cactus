Signal Bot Registration

2024-11-13
1 minute read
homelab
signal , bot

Prepare environment

1sudo apt install docker.io
2sudo systemctl start docker
3sudo systemctl enable docker
4sudo apt install docker-compose
5git clone https://github.com/bbernhard/signal-cli-rest-api.git
6cd signal-cli-rest-api
7docker-compose up -d
8sudo docker update --restart always signal-cli-rest-api-signal-cli-rest-api-1

Register number

1.	Go to Signal Captcha Generator.
2.	Complete the CAPTCHA as instructed.
3.	Right-click on the “Open Signal” link and select “Copy URL.”
4.	Extract the token from the URL. It will look something Step 2: Solve the CAPTCHA

https://signalcaptchas.org/registration/generate
